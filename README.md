# Excel Sales Dashboard

## Project Overview
This project showcases an **interactive Excel Sales Dashboard** built using **Pivot Tables, Slicers, and Pivot Charts** to analyze sales by category, gender, location, and discounts. The dashboard provides key insights for business decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Key Features
- **KPIs**: Total Sales, Average Sale, Total Transactions  
- **Category-wise Analysis**: Compare sales across product categories  
- **Location-wise Analysis**: Analyze sales by city  
- **Gender-based Distribution**: Male vs Female customer sales  
- **Discount Analysis**: Compare discounted vs non-discounted sales  

## Key Insights
- Electronics category contributes the highest sales  
- Delhi and Bangalore are the top-performing cities  
- Discounted orders generate higher net revenue
